This project is submited by Amin Mhana.

this react project uses Redux.
it also uses React Hooks in all its components (no class based components used).

external libraries used for styling and additional functionality:
1.reactstrap (bootstrap styling library for react)
2.fontawesome libraries
3.react-toastify (introduces easy way to present popups in the website)
there are more small dependancies, please refer to package.json for the full dependancy list.

About the Wine Shop website:
wine shop introduces three distinct pages: 1.Signup , 2.Shopping page, 3.Feedback page.
it uses localstorage to store user information.

in order to run project, follow these instructions:
1.under the project path, run cmd => npm install
2.under the project path after npm install run cmd => npm start
